# Alfido Tech Internship Projects

This repository contains the projects completed as part of the **Alfido Tech Data Science Internship**. The internship focused on providing hands-on experience in data cleaning, exploratory data analysis (EDA), machine learning, and business insights using real-world datasets.  

---

## Internship Overview

**Alfido Tech** empowers students and professionals with certified internships, training, and skill-building opportunities in IT, data science, and AI.  
Internship tasks emphasize:

- End-to-end data science workflow: data cleaning → EDA → modeling → reporting
- Building reproducible notebooks with clear documentation
- Hands-on practice with Python, Pandas, NumPy, scikit-learn, Matplotlib, Seaborn, and Jupyter/Colab

**Technologies & Tools Used:**  
Python 3.x | Pandas | NumPy | Matplotlib | Seaborn | scikit-learn | Jupyter / Google Colab

---

## Tasks Completed

### 1️⃣ Zomato Dataset Analysis
- **Dataset:** [Zomato Restaurant Dataset](https://www.kaggle.com/datasets/bhanupratapbiswas/zomato)
- **Goal:** Analyze restaurants, ratings, cuisines, locations, and price factors
- **Key Features:**
  - Data cleaning, missing value handling
  - Exploratory analysis and visualizations
  - Heatmap of correlations
  - WordCloud for popular cuisines
  - 5 business recommendations for Alfido Tech platform
- **Deliverables:** Notebook with cleaned data, visualizations, and PDF report

### 2️⃣ Loan Approval Prediction
- **Goal:** Predict loan approval for applicants using machine learning
- **Key Features:**
  - Data preprocessing and handling missing values
  - Encoding categorical variables
  - Model building using Logistic Regression and Decision Trees
  - Model evaluation using accuracy and confusion matrix
- **Deliverables:** Notebook with code, model results, and insights

### 3️⃣ Instagram Data Analysis
- **Dataset:** [Instagram Posts Dataset](https://www.kaggle.com/datasets/bhanupratapbiswas/instgram)
- **Goal:** Analyze Instagram posts and engagement to optimize posting strategy
- **Key Features:**
  - Parsing timestamps and computing engagement metrics
  - Visualizing best posting times and high-performing hashtags
  - Recommendations for content calendar
- **Deliverables:** Notebook with analysis and visuals, strategy document
